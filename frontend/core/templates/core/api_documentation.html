API Documentation

Base URL: https://yourdomain.com/api/

Overview

This API provides access to the database resources used by the system. It supports searching, retrieving, and downloading datasets.

  • Format: JSON
  • Authentication: Bearer Token (if enabled)

Authentication

Include your API token in the request header:

Authorization: Bearer <your_api_token>

Search Endpoint

GET /api/search/

Parameters

  • q (string, required) — Search keyword
  • limit (integer, optional, default: 20) — Max results
  • offset (integer, optional, default: 0) — Pagination

Example Request

GET /api/search/?q=geneX&limit=10

Example Response

{
  "count": 2,
  "results": [
    {
      "id": 123,
      "name": "GeneX",
      "description": "Protein related to metabolism",
      "created_at": "2025-08-21T10:12:00Z"
    },
    {
      "id": 124,
      "name": "GeneX2",
      "description": "Homolog of GeneX",
      "created_at": "2025-08-21T10:15:00Z"
    }
  ]
}

Errors

400 Bad Request — Missing query parameter
404 Not Found — No results found
